# UAS-TransformasiDigital-6725600014
# Pencegahan Kehilangan Data Akibat Kerusakan Perangkat Menggunakan Backup Google Drive

##  Deskripsi Proyek
Proyek ini membahas penerapan **Google Drive** sebagai solusi penyimpanan berbasis cloud untuk mencegah kehilangan data akibat kerusakan perangkat seperti laptop, komputer, atau smartphone. Dengan melakukan backup secara rutin, data penting tetap dapat diakses dan dipulihkan kapan saja selama tersedia koneksi internet dan akun Google.

---

##  Latar Belakang
Kehilangan data merupakan salah satu risiko yang sering terjadi akibat:
- Kerusakan hard disk atau SSD.
- Laptop atau komputer mengalami error.
- Serangan virus atau malware.
- Kehilangan atau pencurian perangkat.
- Kesalahan pengguna (human error), seperti menghapus file secara tidak sengaja.

Google Drive menyediakan layanan penyimpanan cloud yang dapat digunakan sebagai media backup sehingga data tetap aman meskipun perangkat mengalami kerusakan.

---

##  Tujuan
- Mencegah kehilangan data penting akibat kerusakan perangkat.
- Menyimpan salinan (backup) data secara aman di cloud.
- Memudahkan proses pemulihan (restore) data.
- Meningkatkan keamanan dan ketersediaan data.

##  Langkah-Langkah Backup ke Google Drive

1. Login ke akun Google.
2. Buka **Google Drive**.
3. Klik tombol **New**.
4. Pilih **File Upload** atau **Folder Upload**.
5. Pilih file yang ingin dibackup.
6. Tunggu proses upload selesai.
7. Pastikan file telah muncul di Google Drive.

---

##  Cara Restore Data

1. Login ke Google Drive.
2. Cari file yang ingin dipulihkan.
3. Klik kanan pada file.
4. Pilih **Download**.
5. Simpan kembali ke perangkat.

---

##  Kelebihan Google Drive

- Penyimpanan cloud yang mudah digunakan.
- Sinkronisasi otomatis.
- Data dapat diakses dari berbagai perangkat.
- Mendukung kolaborasi.
- Tersedia riwayat versi file.
- Gratis hingga 15 GB.

---

##  Kekurangan

- Membutuhkan koneksi internet.
- Kapasitas gratis terbatas.
- Kecepatan upload bergantung pada internet.
- Bergantung pada keamanan akun Google.

---

## Tips Keamanan

- Gunakan password yang kuat.
- Aktifkan **Two-Factor Authentication (2FA)**.
- Jangan membagikan akun kepada orang lain.
- Lakukan backup secara berkala.
- Periksa aktivitas login akun Google secara rutin.

##  Hasil yang Diharapkan

Dengan menggunakan Google Drive sebagai media backup:

- Risiko kehilangan data dapat diminimalkan.
- Data tetap tersedia meskipun perangkat rusak.
- Proses pemulihan data menjadi lebih cepat.
- Keamanan penyimpanan data meningkat.
